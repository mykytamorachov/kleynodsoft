Analyzing of the Websites Security

against to the unauthorized access (without TEMPEST) and corrective

We offer

External security analysis:

  • Analysis of web site;
  • Analysis of the database, with can be accessed from the Internet;
  • Analysis of the possibility of unauthorized access to private resources, local area network (LAN).

Internal security analysis:

  • Analysis / creating security policies (document defined user rights - who accessed what and what time) in terms of its correctness;
  • Comparison of the current status of the LAN security policy;
  • Analysis of opportunities of unauthorized access to privet LAN.

The report includes the following sections:

  • Abstract;
  • The purpose of the study;
  • Input;
  • Summary (Executive Summary for management);
  • Technical Report;
  • Recommendations for the vulnerability;
  • General Guidelines;
  • Description of the study;
  • The terms and definitions;
  • Applications.

We offer the following composition of guidelines:

  • for configuring existing tools (firewall, content management systems, scripts control, access-mails and so on);
  • involving additional tools, based on the available capacity of the Customer (patches, new versions, the use of electronic digital signatures, secure connections (SSL, TLS), a persistent information security, platforms, etc.);
  • preparation and correction of the security policy.
